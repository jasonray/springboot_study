### Info

This directory contains a basic springboot javaFx project replica of
[ruslanys/sample-spring-boot-javafx](https://github.com/ruslanys/sample-spring-boot-javafx)

### Run application

Compile and start appplication
```sh
mvn clean spring-boot:run
```
