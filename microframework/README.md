### Info

This directory contains a skeleton spring/flux project based on
[alek-sys/spring-functional-microframework](https://github.com/alek-sys/spring-functional-microframework)

compile and run application
```cmd
mvn clean spring-boot:run
```
Test it via
```sh
http://127.0.0.1:8180/
http://127.0.0.1:8180/json
```
